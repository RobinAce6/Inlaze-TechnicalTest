Ejecución Test Cases en Selenium IDE

- Ir a Chorme Web Store para instalar la extensión de Selenium IDE
https://chromewebstore.google.com/detail/selenium-ide/mooikfkahbdckldjjndioackbalphokd

- Añadir a extensiones de Google.

- Abrir la extensión:

- Presionar Ctrl + O para abrir el proyecto (o en la esquina superior derecha, dar click en el ícono de Carpeta (Open Project))

- Seleccionar el archivo QA-Inlaze-SeleniumIDE.side

- Todos los Test Cases diseñados se mostrarán. 

- Para configurar la velocidad a la que se ejecutará la automatización, click en el ícono de Cronómetro y mover el indicador hacia abajo.

- Clic en el primer ícono Play para ejecutar todos los casos de la Test Suite.
- Clic en el segundo ícono Play para ejecutar el Test Case actual.
